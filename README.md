# 🌐 Business Chatbot Demo (HTML/CSS/JavaScript)

A standalone, interactive AI chatbot UI built for client demos and presentations — no backend required. Demonstrates a realistic chatbot conversation flow with typing animation and quick-reply buttons.

---

## ✨ Features

- 💬 Realistic chat interface with avatar, header, and timestamps
- ⌨️ Typing indicator animation before bot responses
- 🔘 Quick reply buttons (Place order, Talk to a human, View menu)
- 🎨 Clean, modern UI — fully responsive
- ⚡ Zero dependencies — pure HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom animations, flexbox layout)
- Vanilla JavaScript (no frameworks)

---

## 🚀 How to Run

Just open `index.html` in any browser — no build step, no server required.

```bash
git clone https://github.com/Aroojfatima-Dev/business-chatbot-demo.git
cd business-chatbot-demo
open index.html
```

---

## 💡 Use Case

This demo is used to show prospective Fiverr/freelance clients exactly what a custom AI chatbot for their business could look like — before any backend integration work begins. It simulates real conversation flow so non-technical clients can visualize the end product.
